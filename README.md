Omar is an AI-powered assistant specialized in Moroccan traffic law and administrative regulations.
It provides accurate, clear, and practical guidance on fines, penalties, point systems, and legal procedures
try Omar by using this link : https://omar-c2r.lovable.app 

The system is designed for high-compliance environments and ensures consistent, reliable, and language-controlled responses.

Features

Explains Moroccan traffic laws in clear and simple terms

Provides information on fines, penalties, and driving points

Supports administrative procedures and documentation

Enforces strict language compliance (EN / FR / AR)

Optimized for production environments

Easy to integrate into existing systems

Technologies and Tools

Omar is built using modern AI and no-code/low-code technologies, including:

Cohere Large Language Model (LLM) for natural language understanding and response generation

Cohere Embeddings for semantic representation and retrieval

Retrieval-Augmented Generation (RAG) for grounded legal responses

n8n for workflow automation and orchestration

Pinecone Vector Database for semantic search and document indexing

Lovable for frontend interface development

REST APIs for system communication

JSON Configuration for prompt and behavior control

System Architecture
User → Lovable Frontend → API Layer → n8n Workflows
                                 ↓
                         RAG Engine + Pinecone
                                 ↓
                            Cohere LLM
